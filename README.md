## 🔍 Features
- **AI Signature Detection** — Identifies generative AI fingerprints.
- **Geospatial Integrity** — Validates GPS coordinates against physical boundaries.
- **Temporal Forensics** — Exposes timestamp tampering and post-processing.
- **3D Globe Visualization** — Interactive CesiumJS globe for evidence mapping.
